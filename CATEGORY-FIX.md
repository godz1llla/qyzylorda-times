# Исправление категорий - краткая инструкция

## Проблема

Кнопки категорий в навигации не работали - показывали fallback вместо реальных slug.

## Что было исправлено

### ✅ HomeController.php
**Добавлено:** передача `$categories` в view главной страницы

```php
// Получаем категории для навигации
$categories = $this->categoryModel->getAllCategories($lang);

// Передаем в view
$this->view('home/index', [
    // ... другие данные
    'categories' => $categories  // ДОБАВЛЕНО
]);
```

## Как теперь работает

1. **Клик на категорию** → переход по правильному slug
   - "АУЫЛ ШАРУАШЫЛЫҒЫ" → `/auyl-sharuashylygy`
   - "ЖАҢАЛЫҚТАР" → `/zhanalyqtar`

2. **CategoryController::show($slug)** → загружает посты категории

3. **Отображается** → список новостей категории

## Тестирование

Обновите главную страницу http://localhost:8000 и кликните на любую категорию в меню - должна открыться страница с новостями этой категории.
